Calculate Card App
==================

Try https://nitoyon.github.io/1-10calc/

How to build
------------

1. Install Node.js.

2. Install vue cli.

   ```
   $ npm install -g @vue/cli
   ```

3. Install npm packages.

   ```
   $ npm install
   ```

4. Run npm script.

   ```
   # build
   $ npm run-script build

   # start server
   $ npm run-script serve

   # lint
   $ npm run-script lint
   ```
